# 선택자 기본 01

1. tag 선택자 (tag selector)
2. id 선택자 (id selector)   
: tag 선택자 보다 우선한다.   
: 한 html 문서에서 id 선택자는 중복사용 할 수 없다.   
: 명확한 선택을 할 수 있다.
3. class 선택자 (class selector)   
: tag 선택자 보다 우선한다.   
: 한 html 문서에서 중복사용 할 수 있다.
4. 자식 선택자 A>B   
: 직계 자식만을 선택할 때 쓰는 선택자.   
: 드롭다운 메뉴 구성할 때 꼭 쓰이므로 잘 익혀두자.
5. 후손 선택자 A B   
: 자식 및 그 이하 후손을 선택하는 선택자.   
: 일반적으로 많이 쓰이는 선택자이다.
